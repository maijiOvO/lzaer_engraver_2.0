# 训练图像准入标准

> 版本: 1.0  
> 日期: 2026-06-17  
> 适用范围: `dev_tools/test_imgs/` 下所有用于训练预测器的图片

---

## 一、硬性技术指标

| 维度 | 标准 | 说明 |
|------|------|------|
| **色彩** | 必须为彩色（RGB 3通道） | 禁止灰度图、黑白图；即便文件是3通道但三通道相等也视为灰度 |
| **分辨率** | 任一边 ≥1920 且另一边 ≥1090（总像素 ≥2.1MP） | 横图竖图均可；典型横图需 w≥1920, h≥1090 |
| **格式** | `.jpg` / `.jpeg` / `.png` / `.webp` | — |

## 二、语义内容要求

| 维度 | 标准 | 说明 |
|------|------|------|
| **主题** | 城市风景（skyline / 建筑群 / 街景） | 必须包含可辨识的建筑结构、街道、桥梁等城市元素 |
| **光线** | 充足的自然光或人工照明 | 禁止严重欠曝、剪影式逆光导致建筑细节不可辨 |
| **细节密度** | 图中存在可作切割依据的结构特征 | 建筑边缘、窗户纹理、桥梁钢索、街道网格等 |

## 三、明确禁止

| 类型 | 原因 |
|------|------|
| 夜景中建筑结构不可辨的图 | 深度估计器无法从低光区域提取可靠特征 |
| 纯自然风景（山林、湖泊、沙漠） | 缺少城市建筑语义 |
| 室内图、静物、抽象画 | 与城市风景语义不符 |
| 低分辨率缩略图或过度压缩的图 | 细节丢失，切割质量差 |
| 黑白/复古/棕色调艺术照 | 色彩信息丢失 |

## 四、训练/验证分离

图片按 **物理目录** 分为两个集合：

| 目录 | 用途 | 参与训练 | 数量 |
|------|------|----------|------|
| `test_imgs/train/` | 训练集 | ✅ 训练 RandomForest 预测器 | 约 80% |
| `test_imgs/val/` | 验证集 | ❌ 仅评估泛化能力（R² / MAE） | 约 20% |

**规则**：
- 新图下载后先放入 `train/`，标定后由开发者决定是否移入 `val/`
- `val/` 中的图片也需要标定（有 params + features），但不参与训练
- 训练时自动打印：OOB R²（训练集内部）vs 验证 R² / MAE（泛化能力）

## 五、审核流程

1. 图片放入 `dev_tools/test_imgs/` 后，用 **scan 模式** 批量分割
2. **不逐张打断**——全部跑完后统一审核
3. 在 Labeler（`http://localhost:8090`）中查看叠加图和评分
4. 不合格的图直接删除，不标定
