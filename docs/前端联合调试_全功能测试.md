# 前端联合调试 — 全功能测试方案

> **版本:** 1.1 | **日期:** 2026-06-12 | **状态:** ✅ L3.5 手动验证完成（3 个问题已修复：案例 19-21）
>
> 本文档定义前端用户可见功能的完整测试方案。以用户体验为优先，覆盖所有按钮、手势、视图模式、图层控制、导航与错误恢复。Agent 负责 L1(API)/L2(代码+网络采样)，用户负责 L3.5(浏览器手动视觉验证)。

---

## 验证手段说明

| 层 | 含义 | 执行者 | 方法 |
|----|------|--------|------|
| **L1** | API 验证 | Agent | curl 后端/Vite-proxy 确认端点正常 |
| **L2** | 代码+网络 | Agent | grep onClick handler、disabled 逻辑、Network 请求数 |
| **L3.5** | 浏览器手动 | 用户 | 打开 http://localhost:5173 逐项操作验证 |

**手势验证为全局重点怀疑区**：每个涉及 Canvas 预览的步骤，必须独立验证缩放/拖拽/双击复位。
